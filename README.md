# z-skull-breaker
Kill le Z en masse, when they piss me off.

Small game about killing zombies.

# How to Play:

## Start:
use `Z-Breaker.exe` on windows, and `z-breaker.sh` on superior OS

## commandes

click (left, right or middle) to fire.

`wasd`, or arrow keys to move.
